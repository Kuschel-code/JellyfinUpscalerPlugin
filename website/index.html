Features

Everything you need.
Nothing you don't.

A complete AI upscaling pipeline that runs quietly alongside your Jellyfin server.

Docker Microservice

AI processing runs in an isolated container ‚Äî no DLL conflicts, no crashes. Only 1.6 MB plugin vs. 417 MB before.

SSH Remote Transcoding

Offload FFmpeg to GPU-accelerated Docker containers via SSH. Your NAS delegates transcoding to powerful hardware.

5 GPU Architectures

Native support for NVIDIA CUDA, AMD ROCm, Intel OpenVINO, Apple Silicon, and multi-threaded CPU.

Neural Network Models

Seamless Jellyfin Integration

Get Started

Up and running
in minutes.

01

Start Docker Container

Pull and run the image that matches your GPU.

docker run -d \
  --name jellyfin-ai-upscaler \
  --gpus all \
  -p 5000:5000 -p 2222:22 \
  -v /media:/media:ro \
  kuscheltier/jellyfin-ai-upscaler:1.5.1
docker run -d \
  --name jellyfin-ai-upscaler \
  --device=/dev/kfd --device=/dev/dri \
  --group-add video \
  -p 5000:5000 -p 2222:22 \
  -v /media:/media:ro \
  kuscheltier/jellyfin-ai-upscaler:1.5.1-amd
docker run -d \
  --name jellyfin-ai-upscaler \
  --device=/dev/dri:/dev/dri \
  -p 5000:5000 -p 2222:22 \
  -v /media:/media:ro \
  kuscheltier/jellyfin-ai-upscaler:1.5.1-intel
docker run -d \
  --name jellyfin-ai-upscaler \
  -p 5000:5000 -p 2222:22 \
  -v /media:/media:ro \
  kuscheltier/jellyfin-ai-upscaler:1.5.1-cpu
