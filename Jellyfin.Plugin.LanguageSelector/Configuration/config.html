Language Selector Plugin

This plugin adds one-click language selection buttons to episode detail pages.

Features:

  • Automatic detection of available audio and subtitle tracks
  • Flag buttons for quick language selection
  • AniWorld-inspired UI design
  • One-click playback with pre-selected audio/subtitle tracks

Usage:

Navigate to any episode detail page. If multiple language options are detected, flag buttons will appear below the play button.

Supported Combinations:

  • German Flag: German audio, no subtitles
  • JP/DE Flag: Japanese audio + German subtitles
  • JP/EN Flag: Japanese audio + English subtitles
  • US Flag: English audio, no subtitles
